Facebook에서 메시지를 사용하는 것에 관한 도움말입니다. Messenger 앱이나 messenger.com과 관련하여 도움이 필요하면 Messenger 고객 센터를 방문하세요.
Facebook에서 영상 통화 시 카메라, 마이크 또는 스피커가 제대로 작동하지 않는 경우 다음 방법을 시도해보세요.

컴퓨터의 하드웨어 설정 확인

  • 카메라와 마이크가 켜져 있는지 확인합니다. 영상 통화와 같은 프로그램에서 카메라나 마이크가 사용될 수 있으므로 열려 있는 프로그램을 모두 종료합니다.
  • 현재 사용 중인 외부 카메라, 마이크 또는 스피커가 컴퓨터에서 선택되어 있는지 확인합니다.
  • 마이크가 소리 끔으로 설정되어 있지 않은지 확인합니다.
  • 프로그램이 카메라, 마이크 또는 스피커에 액세스할 수 있는지 확인합니다. 컴퓨터나 브라우저 설정에서 확인할 수 있습니다. Microsoft 지원 페이지 또는 Apple 지원에서 컴퓨터 설정을 확인하는 방법을 알아보세요. Google Chrome 지원 페이지에서 Google Chrome 브라우저 설정을 확인해볼 수도 있습니다.

브라우저와 컴퓨터 다시 시작

  • 인터넷 브라우저를 닫고 컴퓨터를 다시 시작합니다.
  • 인터넷에 연결되어 있는지 확인하기
  • 인터넷 연결 속도가 느린 경우 Wi-Fi 네트워크에 다시 연결하거나 다른 네트워크에 연결해보세요.

인터넷 연결 확인

  • 인터넷 연결 속도가 느린 경우 Wi-Fi 네트워크에 다시 연결하거나 다른 네트워크에 연결해보세요.

친구에게 하드웨어와 인터넷 연결을 확인해보라고 요청

  • 영상 통화 중인 사람의 모습이 보이지 않거나 목소리가 들리지 않는 경우 위에 나열된 단계를 따르도록 요청하세요.
